# Pimチャットシステム利用規約

1. 心得  
    
    参加者は大人として振る舞い、他の参加者を大人として扱う。
    
    1. **他人に敬意**を持って接する。
    2. 今は知識がついてわかっている人も最初は初心者、それを忘れず驕らず昂らず。
    3. 何かあっても穏便に対応する。
    4. 怪しいファイルはダウンロードしない。
        ファイル送信者は、ファイルの中身を明記するなどの配慮を行うこと。
    5. モラルハザードを起こさない。
    6. このサーバは幅広い人がいるオープンなコミュニティである。そのため、世界に向けて発言していると考え、発言内容は熟考すること。


2. 禁止行為
    1. **荒らし**行為
    2. 同じ内容の**連投**  
    3. しつこい宣伝  
    4. しつこい**権限**の要求  
    5. 日本国法に抵触する行為  
    6. Discord, Microsoft Teams (for Schools and Students)の規約に抵触する行為  
    7. 外部サービスの規約違反を行う・行ったことを示唆する行為
    8. **宗教勧誘**行為  
        新興宗教等の勧誘などを指し、ソフトウエア、名称等の宗教はこれに含まない。
    9. **反社会的勢力**に該当する行為  
        暴力団・カルト宗教等、反社会勢力にかかわる内容・勧誘。
    10. 企業や学校等の**機密事項**を公に発言する行為  
        守秘義務、インサイダー情報、関係者しか知り得ない情報の投稿。
    11. 特定の人間をタコ殴りにする行為  
        **本人が嫌がっているにも関わらず** 、しつこく進捗を確認し続ける等の行為。
    12. 危険行動を助長する行為  
    13. **薬物乱用**の宣言・助長する行為  
        オーバードーズ、違法薬物に関する内容。
    14. **犯罪行為**の宣言、犯罪行為を助長する行為  
        未成年飲酒及び喫煙、無免許、窃盗、電磁気等改ざん、ウイルス・マルウエア等の配布。  
        ただし、「そういうデータであることを宣言」し、検証素材として提供する場合は除く。
    15. **誹謗中傷**  
    16. **わいせつ図画**を投稿する行為  
        刑法175条 (わいせつ図画) に該当する可能性のある画像の投稿禁止。  
        nsfw 指定のあるチャンネルであっても、**無修正・修正の薄い画像** に関しては投稿を禁止する。  
        nsfw に関しては、別途制限を設ける。下記項目を参照のこと。  

    17. **他人の個人情報を公表**する行為  
        ただし、公開情報及び企業情報は除く。  
        公開情報とは誰でも以下の方法で確認できるものとする。
        - Webサイトや調査システム・Whois等に公開されている  
        - 行政資料等の請求で開示ができる  
        - 本人が公言している場合 (この場合、「投稿している」必要があります。VCでの発言は証拠の検証が不可能であるため、これに含まない。)  
        上記に関しては、情報のソースについて確認する場合がある。  
            
        具体例  
        - 他人の情報に関して「ここで知り得ない」にも関わらず、言及する行為。
        - 他人の写真の本人の意図に沿わない投稿。
        - 個人情報の本人の意図に沿わない投稿。
    18. ボイスチャットの許可を取っていない録音・録画および、内容を参加者以外へ伝える行為  
        - その場所にいない人に、意図せず内容が伝わることで、発言者が何らかの被害を被る可能性があるため
        - 意図せず、発言した内容が不特定多数に伝わることによって、不特定多数に迷惑をかけてしまう可能性があるため
    19. その他**倫理観**が欠如している行為及び、**社会通念を逸脱**する行為  
            給料マウント、しつこい政治活動、疑似科学、反ワクチン活動、モラルハザード 等


3. 注意事項
    1. ファイルを送信する際、そのファイルがどういったものなのかを示す必要がある。
    2. Pim Mattermostではファイル容量制限を8MBとしている。
       これを超えるデータはGoogle Drive等に保存し、リンクを共有する形で送信を行うこと。
       もしどうしてもPim Mattermostに直接アップロードしなければならないファイルの場合、別途相談すること。
    3. テンプレートの使用  
        一部チャンネルでは独自のテンプレートを設置している場合がある。  
        これにより、有用な情報に簡単に遡れるため、できるだけ従うこと。  
        また、それ以外の独自ルールを設けている場合もある。都度ピン止めを確認すること。
    3. チャンネル作成時のルール
        適切にチャンネルの管理ができるよう、チャンネル作成時の命名規則が明確に示されている。
        詳細は別紙を確認すること。これに従わないチャンネルはシステム管理者がチャンネル名を予告なしで変更する場合がある。
    4. 私的利用について
        Pim Mattermostの運用はOBの方をはじめとする寄付によって賄われている。
        そのため、Pimの活動に関係が無い使用方法は歓迎されず、私的利用されたチャンネルやデータ保持は保証せず、
        場合によっては削除を行う場合もある。


4. R18, R15制限に関して
    1. ゾーニング  
        - **性器及び性行為**に類する画像の貼り付けは nfswを指定したチャンネルで行うこと。
            - チャンネル名に`nonsafe`を含むチャンネルは、nsfw指定されたとみなされる。
        - 年齢の範囲も広いため、**R15相当** (女性の胸に関わる) 部分の露骨な表現に関しては **nsfw** とする。
        - 「性的目的で利用される製品」に関しても nsfw 指定とする。
        - **露骨な表現及び無修正を含む画像に関しては投稿を禁止**する。
        
        参考情報: 男女ともに水着で隠れる部分が見えている画像に関しては nsfw 指定とする。
    2. R18 URL の貼り付けに関して  
        nsfw 指定チャンネル以外では、リンクやスポイラーを含めたすべてのNSFWコンテンツの投稿を禁ずる。  
        当該投稿は削除対象となる。

        ただし、当該投稿が埋め込みリンクを無効化する方法で記述され、NSFWなコンテンツであることが明示されている場合、
        軽減措置としてこれを投稿できる。
    3. グロテスク表現のゾーニング 
        参加者の年齢幅が非常に広く、また、グロテスク表現に嫌悪感を抱く方がいることを考慮し、下記のように運用する。  
        - グロテスク表現に関しては、 nsfw を指定したチャンネルで行う。


5. 闘争行為 に関して  
    **過熱した口論**は禁止する。過熱した口論を目撃した場合、両者のアカウントを一時的に無効化し、クールダウンを設定することがある。
    - 宗教戦争や派閥争いを助長する行為は禁止される。
    - 過熱した口論とは、相手を明らかに尊重していない発言等が多数見られる状態を示す。


6. 権限  
    1. システム管理者
        Mattermostシステムを管理している、もしくはシステム管理チームの監査を行うユーザが所有する。
        モデレータを兼任し、これらのユーザはすべてのチームとチャンネルにおいて不適切なメッセージの削除と検閲を行う。
    2. チーム管理者
        Pimチームや外部との連携チーム等のあらかじめ指定された限られたチームにおいて、モデレーションの権限が付与される。
    3. チャンネル管理者 
        特定のチャンネルに対してモデレーションの権限が付与されるユーザ。
        チャンネルを作成した際、作成したチャンネルに対して自動で付与される。ひとりごとチャンネル（Times channel）には必ず割り当てられる。

7. 依頼  
    チャンネルの追加、機能の追加、発言の削除、荒らしの対応等、何らかの要望がある場合は ~運営に訊く にて要望を送ることが可能である。  

    ひとりごとチャンネルに関しては、下記の操作を使用者の希望に応じて行う。
    - webhook の権限  
    - チャンネル名の変更  
    - チャンネルのアーカイブ・削除


8. 処罰  
    次の場合、システム管理者が各種権限の剥奪・閲覧制限等を行う場合がある。  
    - 禁止行為に触れる行為を行った場合。  
    - モデレーターが必要とした場合。  
    また、必要であるとモデレータが判断した場合はPim 運営メンバーや必要に応じて大学や公的機関を含め、会議[1](#note1)を行い、タイムアウト、アカウント無効化等の処罰を行う場合がある。


9. 荒らしの緊急対応  
    システム管理者は、緊急であると判断した場合、一存でアカウントの無効化を行うことができる。  
    モデレータはタイムアウトを設定し、正副管理者に報告すること。  
    メンバーはDiscord上の#障害情報チャンネルで `@sysadmin` を行い、
    システム管理者に対応を促すこと。


10. 免責
    - 当サーバー以外での利用者同士のトラブル等について、島根大学ものづくり部 Pimおよび島根大学は一切関与をしないものとする。
    - 当サーバーにて起こった情報流出やデータ破損等のすべての出来事について、島根大学ものづくり部 Pimおよび島根大学は一切の責任を負わない。


これは、 Pim運営メンバー に対し多大な迷惑をかける行為である。
